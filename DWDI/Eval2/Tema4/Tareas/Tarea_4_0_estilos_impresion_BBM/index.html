Página con Estilo Personalizado

Subtítulo de la Página

Esta es una página de ejemplo que utiliza una fuente personalizada aplicada a todos los elementos del documento. Además, incluye estilos diferentes para impresión, con una configuración específica para un documento en tamaño A5 y orientación horizontal. Aquí demostramos cómo se puede adaptar el diseño de un sitio web para diferentes medios.
